S32K144 Dual Node CAN Communication System

Overview:
	This project is designed to demonstrate CAN communication between two independent nodes. The hardware selected for this project is NXP S32K144EVB microcontrollers and it demonstrates real time communication between the nodes
and event driven behaviour.

Improvements Made:
	-Convert LED control to fully non blocking and event driven
	-Expanded to handle multiple CAN commands

Near Future Improvements: 
	-Implementing physical actuators and buttons for control tower operation
	-Integrate RTOS for task based scheduling example
